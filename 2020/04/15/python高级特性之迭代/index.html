python 高级特性之迭代

给定一个list或者tuple,通过for循环遍历整个list或者tuple，这种遍历我们称之为迭代
python中，我们使用for......in来进行迭代。（很多语言使用下标来完成。）
python的for循环抽象成都要高于C的for循环，因为在python中for循环不仅仅用在list或者tuple上，还可以作用于其他的迭代对象之上。
list这种数据类型虽然有下标，但是很多数据类型是没有下标的（list数据下标从0开始），但是，只要是可迭代对象，无论有无下标，都可以迭代。
那么，什么是可迭代对象？简单来讲，可以被for循环迭代的对象就是可迭代对象。那么，如何判断一个对象是可迭代对象呢？方法是通过collections模块的Iterable类型判断：

至于怎么定义一个对象是可以迭代的呢？在数据类型中，定义函数以满足for函数的要求，这种要求称之为“协议”。迭代所需要满足的协议成为可迭代协议。可迭代协议在内部实现了__iter__()方法，一般来说，如果对象中有__iter__()方法，这个对象就是可迭代对象。

默认情况下，dict迭代的是key。如果要迭代value，可以用for value in d.values()，如果要同时迭代key和value，可以用for k, v in d.items()。
